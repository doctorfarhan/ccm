## Overview

CCM Notes is a collection of quick-reference protocol cards and comprehensive clinical notes covering essential topics in intensive care medicine. All content is evidence-based, aligned with current guidelines (SSC, AHA ACLS 2020, ESC, KDIGO), and designed for rapid bedside consultation.

## Features

- **20 Protocol Cards** — Visual quick-reference cards for common ICU scenarios
- **Clinical Notes** — Detailed, evidence-based drug references with dosing tables and trial evidence
- **Mobile Responsive** — Fully optimized for phones, tablets, and desktop
- **Offline Ready** — Single HTML files, no external dependencies
- **Print Friendly** — Clean layouts suitable for printing or PDF export
